Development server required

You are opening the source HTML directly or the server is serving raw JSX files with an unsupported MIME type. Start the dev server or build the project and serve the dist directory.

# macOS (Homebrew)
brew install node
corepack enable
corepack prepare pnpm@latest --activate
pnpm install
pnpm dev
-- or --
pnpm build
python3 -m http.server --directory dist 3000
